# account_book.py
# 简易收支记账本
## 项目文件结构
account-project/
├── README.md        # 项目说明文档（本文件）
├── account_book.py  # 主程序源码
└── account.txt      # 自动生成，存储收支账单数据，分隔符 |

## 使用说明
运行 `account_book.py`，根据菜单数字选择对应功能操作。
录入账单时，日期项直接回车会自动填充本地当天日期，无需手动输入。

## 数据存储规则
1. 本地自动生成 `account.txt` 文件保存所有收支账单
2. 数据分隔符：`|`，单行存储格式：`日期|收支类型|金额|备注`
3. 程序启动自动加载历史账单；增删操作自动同步写入文件

## 核心新学习知识点
- 收支双类型业务逻辑，区分收入与支出独立统计
- 结余计算公式：总收入 - 总支出 = 当前结余
- 列表嵌套字典存储多字段账单记录
- `sum()` 条件筛选求和：按收支类型分别统计金额
- `:.2f` 格式化输出，金额统一保留两位小数
- 按序号删除列表元素：序号与列表下标的转换
- `datetime` 模块获取本地系统日期，自动填充默认值
- 空输入判断实现默认值，提升录入效率
- 浮点数输入校验 + 金额正数合法性判断
